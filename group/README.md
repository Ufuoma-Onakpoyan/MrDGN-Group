# MR DGN Group Website

Main holding company website for MR DGN Group - overseeing MR DGN Entertainment, MR DGN Construction, and MansaLuxe Realty.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Setup

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Media page video

The Media page shows a company overview video. To add it: place your MP4 file at `public/assets/MR DGN Group.mp4`. See `public/assets/README.md` for details.
